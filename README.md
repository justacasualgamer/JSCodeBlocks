# JSCodeBlock
## Note: This extension only works in Chrome.
## Note 2: The extension has an icon (knitgh.png). It's a Hollow Knight reference :)
My first Chrome extension about JavaScript. You can create code blocks and save them to execute on other websites.
# How to install
Click on 'Code', then 'Download ZIP file'. Extract them to a folder, then go to chrome://extensions/, turn on Developer mode, select 'Load unpacked' and select the folder you just extracted to add the extension.
